For checking case, Python also offers:

.isupper(): Returns True if all cased characters are uppercase. 
.islower(): Returns True if all cased characters are lowercase. 
.istitle(): Returns True if the string is in title case. 
These methods return new strings and do not modify the original string in place.  For case-insensitive comparisons, .casefold() is often recommended over .lower() as it is more aggressive and handles more Unicode characters correctly.
