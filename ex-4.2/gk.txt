1. So i want print with quotes (i.e. i was said "Don't panic python is easy" )
Ans. Here you can use ( \ ) back slash before the word you want to print quotes and after word ends use the same quotes and then ( \ ).
     Another method is if you want to print double quotes on screen the start with ( ' ) single quotes before first word and the word you want to get print use ( " ) double quotes before that word start and after the desire word complete use ( " ) double quotes and after the very last word of the sentence use ( ' ) single quotes again to close the string.
       If you want to print quotes then just exchange ( ' ) single quotes with ( " ) double quotes.
 Note : You can get an example from our ( quotes-1.py ) file.


2. I want to print a whole paragraph in normal style. How can i do that ?
Ans. Just add an \ after a line ends and go on ( this style helps if you simply prints without preserving white spaces ).
     Another method is before your paragraph start use ( """/''' ) triple quotes at onece and same ( """/''' ) triple quotes in after last word ( this will preserve white spaces mean this will print as it code ).
